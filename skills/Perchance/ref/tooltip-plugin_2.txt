<h1>💬&#xFE0E; Tooltip Plugin 💬&#xFE0E;</h1>

<div style="margin:0 auto;width:100%;max-width:650px;background:white;border-radius:2px;padding:1em; box-sizing:border-box; text-align:left;">
	<p>Put this in your Perchance code panel:</p>
	<pre>tooltip = \{import:tooltip-plugin\}</pre>
	
  <p>Now you can create tooltips:</p>
	<pre>\[tooltip("ℹ️", "tooltip text!")\]</pre>

	<p>The above example would result in this (try hovering over it):</p>
	<p style="text-align:center;">[$output("ℹ️", "tooltip text!", {interactive:true})]</p>
	
	<p>So, as you can see, the format is:</p>
	<pre>\[tooltip(anchorText, tooltipText)\]</pre>
	<br>
  <p>Here's an example of how you might want to use it:</p>
<pre>
output
  There is &lt;b>\[tooltip(race, description)\]&lt;/b> standing nearby.

race
  an orc
  an elf
  a human
  
description
  They look suspicious.
  Their shirt is torn.
</pre>
  <p>And you's get something like this:</p>
  <p style="text-align:center; background:#f6f6f6; padding:1rem;">There is <b>[$output("an orc", "Their shirt is torn")]</b> standing nearby.</p>
  <br>
  <p>You can customize the background color and the text color like so:</p>
<pre>
tooltipOptions
  css = background-color:tomato; color:yellow;
  ...

output 
  Try &lt;b>\[tooltip("hovering over this text", "Here it is!", tooltipOptions)\]&lt;/b> to see a custom-colored popup. Random num: \{1-100000\}
</pre>
  <p>You can replace "tomato" and "yellow" with <a href="https://www.w3schools.com/cssref/css_colors.asp" target="_blank">any valid CSS color</a>, including completely custom <a href="https://www.google.com/search?q=hex+color+picker" target="_blank">hex codes</a> like "#32a852". <a href="https://perchance.org/tooltip-plugin-custom-colors-example#edit">Here's an example</a> which uses the above code.</p>
  <p>If you'd like to include links (or other HTML) in the popup, you need to use these options:</p>
<pre>
tooltipOptions
  interactive = true
  allowHTML = true
  ...
  
output
  \[tooltip(anchorText, tooltipText, tooltipOptions)\]
</pre>
  <p>The <code>interactive</code> option makes it so the popup doesn't dissapear when you try to move your mouse into it (e.g. to click a link that is within the popup/tooltip). The <code>allowHTML</code> option allows us to put HTML in the tooltip, so with these two options we could include a link within the tooltip like so:</p>
<pre>
tooltipOptions
  interactive = true  
  allowHTML = true

tooltipText
  &lt;a href\="https://i.imgur.com/26DZgXM.mp4" style\="color:white;">Here's&lt;/a> the secret link. And the secret code is \{1-100000000\}

output
  You can \[tooltip("hover over this text", tooltipText, tooltipOptions)\] to get the secret link.
</pre>
  <p><a href="https://perchance.org/tooltip-plugin-options-example#edit">Here's</a> an example which uses that code.</p>
  <p>All customization options are listed on the <a href="https://atomiks.github.io/tippyjs/v6/all-props/">tippyjs homepage</a>.</p>
