<h1>Select Range Plugin</h1>

<div style="margin:0 auto;width:100%;max-width:780px;background:white;border-radius:2px;padding:1em; box-sizing:border-box;">
	<p>Put this in your Perchance code panel:</p>
	<p><code>selectRange = \{import:select-range-plugin\}</code></p>

	<p>Now you select specific ranges of items from your list:</p>
	<ul>
		<li>3rd to 7th item: <code>\[selectRange(animal, 3, 7)\]</code></li>
		<li>12th to 30th item: <code>\[selectRange(animal, 12, 30)\]</code></li>
		<li>4th item onwards: <code>\[selectRange(animal, 4)\]</code></li>
		<li>26th item onwards: <code>\[selectRange(animal, 26)\]</code></li>
	</ul>
	<p>So the notation is <code>\[selectRange(list, start, end)\]</code>.</p>
	<p>If you omit <code>end</code>, it'll assume you want all the rest of the items.</p>
	<p>Both numbers in the range are <b>inclusive</b> (start and end items are included).</p>
	<p>You can also reference items counting <b>from the end of the list</b> using the minus sign:</p>
	<ul>
		<li>last 5 items: <code>\[selectRange(animal, -5)\]</code></li>
		<li>only the last item: <code>\[selectRange(animal, -1)\]</code></li>
		<li>last 3 items: <code>\[selectRange(animal, -3)\]</code></li>
		<li>2nd item up to the 4th last item: <code>\[selectRange(animal, 2, -4)\]</code></li>
		<li>5th last item up to the 3th last item: <code>\[selectRange(animal, -5, -3)\]</code></li>
		<li>all items except the last one: <code>\[selectRange(animal, 1, -2)\]</code></li>
		<li>all items except the last 3: <code>\[selectRange(animal, 1, -4)\]</code></li>
	</ul>
	<p>You can use the "sublist" like it's a normal list:</p>
	<ul>
		<li><code>\[sublist = selectRange(animal, 3, 8), ""\]</code> and then later:</li>
		<li><code>\[sublist.pluralForm\]</code>, or:</li>
		<li><code>\[sublist.joinItems(",")\]</code>, or:</li>
		<li><code>\[selectRange(sublist, 3, 5)\]</code></li>
		<li>etc.</li>
	</ul>
	<br>
	<p><b>Notes:</b></p>

	</ul>
