<h1>URL Params Plugin</h1>

<div style="margin:0 auto;width:100%;max-width:750px;background:white;border-radius:2px;padding:1em; box-sizing:border-box; text-align:left;">
	<p style="margin-top:0"></p>
  
  <p>To use this plugin, you'll first need to import it by adding this line to your lists editor:</p>
<pre>
url = \{import:url-params-plugin\}
</pre>
  <p>And now try putting this in your lists editor:</p>
<pre>
output
  In this page's current URL, 'blah' is set to \[url.blah || "nothing"\]
</pre>
  <p><a href="https://perchance.org/url-params-plugin-example?blah=abc123#edit" target="_blank">Here's an example generator</a> with that code.</p>
  <p>And you can of course use your URL data like you would with any other data/variables - for example:</p>
<pre>
output
  Here's an animal based on the URL's 'type' parameter: \[animal.selectOne.selectOne\]

animal
  mammal ^\[url.type === "mammal"\]
    mouse
    rabbit
  reptile ^\[url.type === "reptile"\]
    snake
    lizard
</pre>
  <p>Here's an example of using a 'lang' parameter to change the language of your generator</p>
<pre>
output
  Here's a random fruit: \[fruitEn.selectOne\]     ^\[url.lang === "en"\]
  Voici un fruit au hasard: \[fruitFr.selectOne\]  ^\[url.lang === "fr"\]

fruitEn
  apple
  banana

fruitFr
  pomme
  banane
</pre>
	<p><b>Notes:</b></p>
	<ul>
    <li>Get started <a href="https://perchance.org/url-params-plugin-example?blah=abc123#edit" target="_blank">with this example</a>.</li>
    <li>This can be paired with the <a href="https://perchance.org/seeder-plugin" target="_blank">seeder-plugin</a> to allow you to create unique links to specific generator outputs.</li>
    <li>To add a parameters to a URL, you must add <code>?</code> to the end of the URL, and then a list of <code>name=value</code> separated by <code>&amp;</code>. So, for example, a URL with 3 parameters would look like this: <code style="white-space:nowrap;">https://perchance.org<b>?</b>fruit=apple<b>&amp;</b>animal=mouse<b>&amp;</b>lang=english</code></li>
    <li>Currently you can only 'read'/'get' the URL parameters - you can't change them. If you want to change the current URL, you should instead create a <a href="https://perchance.org/link-plugin" target="_blank">link</a> which the user can click to open the new URL.</li>
    <li>(Note for curious web/JavaScript devs: Even though your generator runs within an iframe, it can 'see' the URL parameters of the parent window because Perchance copies those parameters over to the iframe when initializing it)</li>
    <li>Check out more plugins at <a href="/plugins">perchance.org/plugins</a></li>
	</ul>
</div>
