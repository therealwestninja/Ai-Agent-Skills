<h1>[$output("Font Plugin", "Pacifico", "170%", "#ff5252")]</h1>

<div style="margin:0 auto;width:100%;max-width:690px;background:white;border-radius:2px;padding:1em; box-sizing:border-box;">
	<p>Put this in your Perchance code panel:</p>
	<p><code>font = \{import:font-plugin\}</code></p>
	<br>
	<p>Here we output a random item from our <code>animal</code> list in the "<a href="https://fonts.google.com/specimen/Pacifico">Pacifico</a>" font:</p>
	<p><code>\[font(animal, "Pacifico")\]</code></p>
	<br>
	<p>All font names must be from <a href="https://fonts.google.com">Google Fonts</a> (If you want to use a custom font, <a href="https://perchance.org/custom-font-importer" target="_blank">click here</a>).</p>
  <p>Here's another example to try:</p>
	<p><code>\[font(myList, "Permanent Marker")\]</code></p>
	<p>Make sure you change "myList" to the name of an actual list or variable/property in your generator. If you just want it to output some fixed text you can write this:</p>
	<p><code>\[font("hello there!", "Permanent Marker")\]</code></p>
	<p>which would output this:</p>
	<p>[$output("hello there!", "Permanent Marker")]</p>
	<br>
	<p>You can also change font size and color:</p>
	<p><code>\[font(myList.upperCase, "Courgette", "30px", "#ff5252")\]</code></p>
	<p>Use a <a href="https://www.google.com.au/search?q=hex+color+picker">hex color picker</a> to get custom colors in <code>#XXXXXX</code> format.</p>
	<p>If you want to change the color but not the size, then use 100% for the size to keep the size at the default:</p>
	<p><code>\[font(myList.upperCase, "Courgette", "100%", "#ff5252")\]</code></p>
	<p>You can (for example) write 70% to make the font 70% of the height of the "default" font size:</p>
	<p><code>\[font(myList.upperCase, "Courgette", "70%", "#ff5252")\]</code></p>
	<br>
	<p>What if you want to change the default font of the <b>whole page</b>? Here you go:</p>
	<p><code>\[font(null, "Courgette", "25px", "pink")\]</code></p>
	<p>And to apply the font to a specific HTML element, just give the element an <code>id</code> as shown below, and then put the <code>id</code> as the first value:</p>
<pre class="custom">
&lt;p id="myParagraph"&gt;Hello this is my \[adjective\] paragraph and it has a cool font.&lt;/p&gt;
\[font(myParagraph, "Courgette", "25px", "pink")\]
</pre>
	<br>

	</ul>
