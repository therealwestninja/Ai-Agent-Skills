<h1>🖼️ Favicon Plugin</h1>

<div style="margin:0 auto;width:100%;max-width:750px;background:white;border-radius:2px;padding:1em; box-sizing:border-box; text-align:left;">
  
  <p style="margin-top:0">This plugin allows you to change the little icon that's shown in the browser tab.</p>
  <p>To use this plugin, you'll first need to import it by adding this line to your lists editor:</p>
<pre>
favicon = \{import:favicon-plugin\}
</pre>
  <p>And then put this in the HTML (bottom-right) panel:</p>
<pre>
\[favicon("https://example.com/foo.png")\]
</pre> 
  <p><a href="https://perchance.org/favicon-plugin-example#edit" target="_blank">Here's an example generator</a> so you can see it in action.</p> 
  <p><b style="color:red;">Important:</b> You should upload your favicon images to <a style="font-weight:bold;" href="https://perchance.org/upload" target="_blank">perchance.org/upload</a>. Don't link to a random image on the internet, since the average 'life expectancy' of a random URL on the internet is only a couple of years. Instead, download the image, then upload to <a href="https://perchance.org/upload" target="_blank">perchance.org/upload</a>, and you'll be given a <b>permanent</b> link to that image.</p>
  <p><a href="https://perchance.org/animated-favicon-example#edit" target="_blank">Here's a silly example<a> where we make an "animated" favicon by changing it once per second through a series of animation frames.</p>
	<p><b>Notes:</b></p>
	<ul>
    <li>For mad scientists: You can pass a 'data URL' to this plugin. <a href="https://perchance.org/text-to-image-plugin-programmatic-example#edit" target="_blank">Here's an example</a> that generates an image with the <a href="/text-to-image-plugin">text-to-image-plugin</a>, then set the tab icon to that image. You could also e.g. make a little dot on a blank canvas that moves around based on arrow key presses, and then send a data URL of each frame to this plugin to make a very tiny game that uses the browser tab icon as the "screen".</li>
    <li>Check out more plugins at <a href="/plugins">perchance.org/plugins</a></li>
	</ul>
</div>
