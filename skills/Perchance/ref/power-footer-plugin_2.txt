  <div class="main">
    <p>Simply insert this into the list panel:</p>
    <pre>
footers = \{import:power-footer-plugin\}

footerItems
  This is a footer.
  \{1-1000\}
  \{import:cliche\}
  I have \{a\} \{import:noun\}.</pre>
    <p>And then write this code anywhere in the HTML panel:</p>
    <pre>\[footers(footerItems)\]</pre>
    <p>You will get this:</p>
    <p style="text-align: center">[$output(footerItems)]</p>
    <p>You can insert literally anything you want:</p>
    <pre>
footerItemsAnything
  &lt;span onclick\\="update()">Randomize\{!|.|\}&lt;/span>
  &lt;font color\\="\{import:hex-color\}">The \{import:animal\}&lt;/font>
  &lt;b>&lt;i>&lt;u>Let them do it.&lt;/u>&lt;/i>&lt;/b></pre>
    <p style="text-align: center">[$output(footerItemsAnything)]</p>
    <p>You can also insert links to the items:</p>
    <pre>
footerItemsLinks
  Power Name Generator::link:power-name
  Minecraft Block Generator::link:minecraft-random-block-generator
  An example website::link:https://example.com
  My generators::link:open-in-new-tab:power-generator-manager</pre>
    <p style="text-align: center">[$output(footerItemsLinks)]</p>
    <p>You can customize the background color of the items:</p>
    <pre>\[footers(footerItems, "cyan")\]</pre>
    <p style="text-align: center">[$output(footerItems, "cyan")]</p>
    <pre>\[footers(footerItems, "#80F7EB")\]</pre>
    <p style="text-align: center">[$output(footerItems, "#80F7EB")]</p>
    <p>You can add some style to the footer items:</p>
    <pre>\[footers(footerItems, null, "border: 2px solid blue")\]</pre>
    <p style="text-align: center">[$output(footerItems, null, "border: 2px solid blue")]</p>
    <pre>\[footers(footerItems, null, "color: white; background: linear-gradient(0deg, rgba(99,197,254,1) 0%, rgba(44,82,222,1) 100%)")\]</pre>
    <p style="text-align: center">[$output(footerItems, null, "color: white; background: linear-gradient(0deg, rgba(99,197,254,1) 0%, rgba(44,82,222,1) 100%)")]</p>
    <p>You can even change the color of the footer items that appears on hover:</p>
    <pre>\[footers(footerItems, null, null, "#3F8AFC")\]</pre>
    <p style="text-align: center">[$output(footerItems, null, null, "#3F8AFC")]</p>
    <pre>\[footers(footerItems, null, null, "\{import:hex-color\}")\]</pre>
    <p style="text-align: center">[$output(footerItems, null, null, "{import:hex-color}")]</p>
    <p>You can also change the text color on hover:</p>
    <pre>\[footers(footerItems, null, null, null, "#1E7401")\]</pre>
    <p style="text-align: center">[$output(footerItems, null, null, null, "#1E7401")]</p>
    <pre>\[footers(footerItems, null, null, null, "navy")\]</pre>
    <p style="text-align: center">[$output(footerItems, null, null, null, "navy")]</p>
    <p>[temp("new")] You can now change the text and background colors that appear when the mouse button is clicked on the items:</p>
    <pre>\[footers(footerItems, null, null, null, null, "#28D62E")\]</pre>
    <p style="text-align: center">[$output(footerItems, null, null, null, null, "#28D62E")]</p>
    <pre>\[footers(footerItems, null, null, null, null, null, "mediumBlue")\]</pre>
    <p style="text-align: center">[$output(footerItems, null, null, null, null, null, "mediumBlue")]</p>
    <p>If you want to disable the clicking effect, you can do that too:</p>
    <pre>\[footers(footerItems, null, null, null, null, "none", "none")\]</pre>
    <p style="text-align: center">[$output(footerItems, null, null, null, null, "none", "none")]</p>
