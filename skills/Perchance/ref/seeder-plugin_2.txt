<h1>Seeder Plugin 🌱</h1>

<div style="margin:0 auto;width:100%;max-width:750px;background:white;border-radius:2px;padding:1em; box-sizing:border-box;">
	<p style="margin-top:0;">This plugin allows you to create seeded randomness - so you can give your generator a string, and it'll generate the same "random" sequence of selections if you enter that string later. Just put this code in your perchance code panel:</p>

<pre class="custom">
seeder = \{import:seeder-plugin\}
</pre>

<p>And then put this in your HTML panel:</p>
	
<pre class="custom">
&lt;input placeholder="Enter a seed..." oninput="seeder(this.value), update()"&gt;
</pre>
	
	<p>So now when you type some letters and numbers into that input form, the seed will be updated and all the following "randomize" clicks will produce <u>a sequence that is specific to that particular seed</u>. So you could send the seed to your friend and they would paste it in and each time they clicked the "randomize" button they'd see <i>exactly</i> the same sequence of random results that you saw.</p>
	
  <p>Want to embed the seed in the URL, so you can share your generator with a link that has a default seed embedded? Import the <a href="https://perchance.org/url-params-plugin" target="_blank">url-params-plugin</a> like this:</p>

<pre class="custom">
url = \{import:url-params-plugin\}
</pre>
  <p> and then write <code>\[seeder(url.seed)\]</code> at the top of your HTML (bottom-right) editor panel.</p>
  
  <p>And now you can share your generator with a seed like this:</p>
  <pre>https://perchance.org/your-generator-name?seed=yourseedtext</pre>
  <p>...and the page will load with a default seed of <code>yourseedtext</code>. <a href="https://perchance.org/seed-from-url-example#edit" target="_blank">Here's</a> a working example. And <a href="https://perchance.org/seed-from-url-with-link-example#edit" target="_blank">here's an example</a> that has a live-updating link to the seed as you change it.</p>
  
	<br><br>
	<p><b>Notes:</b></p>
	<ul class="dot-list">
		<li><a href="https://perchance.org/seeder-plugin-example#edit">Here's an example generator</a> that uses this plugin.</li>
		<li><a href="https://perchance.org/use-seeder-plugin-for-one-import-only-example#edit">Here's an example</a> that that shows you how to use the seeder plugin for only one import or list (the other lists stay completely random).</li>
		<li>If you leave the input form empty, then no seed will be used, and you'll instead get completely random selections, just like normal.</li>
		<li>Since we're using the <code>oninput</code> event of the <code>&lt;input&gt;</code> element, the seed is updated at every keypress. In case anyone wants an example where you need to click a "submit" button to update the seed, <a href="https://perchance.org/dimzpj4mqr#edit">here you go</a>. This example uses the second parameter of the seeder function: <code>seeder("my seed", "forceUpdate")</code> which specifies that the seed should be "reset" even if it hasn't been changed.</li>
		<li>Check out more plugins at <a href="/plugins">perchance.org/plugins</a></li>
	</ul>
</div>
