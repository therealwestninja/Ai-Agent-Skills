<h1>Upload Plugin Compression Example</h1>
<p style="font-size:80%; max-width:100%; width:750px; margin:1rem auto;">By compressing data before uploading, you can fit 2-10x more data (depending on the data type and amount of data), so it's often worth doing this.</p>
<p style="font-size:80%; max-width:100%; width:750px; margin:1rem auto;"><b>Note:</b> If some text is already extremely small (e.g. only a few characters), then compression with the 'gzip' format (used here) may actually increase the file size a bit. Generally you wouldn't be uploading a text file with only a few characters though, so it's nothing to worry about. The larger the text, the higher the compression ratio will tend to be.</p>
