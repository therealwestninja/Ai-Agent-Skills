  <h3>Option 1: Using Glitch or Replit</h3>
  <p style="margin-top:0;">Okay so I haven't written a proper tutorial for this yet, but there are some instructions here that shouldn't require any coding knowledge:</p>
  <p><a href="https://glitch.com/edit/#!/perchance-discord-server-bot-template" target="_blank">glitch.com/edit/#!/perchance-discord-server-bot-template</a></p>
  <p>You can play around with a working version of the above bot on the <a href="https://discord.gg/43qAQEVV9a" target="_blank">official Perchance Discord server</a>. Instructions on how to use it are <a href="https://discord.com/channels/970057744612724746/970057745665499148/970317365755654144" target="_blank">here</a>.</p>
  <p>Note that Glitch requires you to pay a monthly fee if you want your bot to stay on. The price is currently $8/month which gives you up to 5 bots/servers that always stay online.</p>
  <h3>Option 2: Self-hosting</h3>
  <p>If you don't want to use Glitch, then you can run your bot on your own computer for free, so long as you're okay with keeping your computer on (your bot will turn off when you turn your computer off), and you know how to install software on your computer, and you can follow the slightly-technical instructions below. Here's what you need to do:</p>
  <ol style="text-align:left;">
    <li>Click the Glitch.com link above and follow the instructions to set up your Glitch server. Check that it's working correctly by running <code>!perch</code> commands on your Discord server.</li>
    <li>Download your Glitch project. As of writing, you need to click the "Tools" button, then "Import / Export", and then "Download project". But the user interface might have changed since I wrote this, and the button labels might have changed, so you may have to hunt around to find it. It should download a <code>.tgz</code> file, or <code>.zip</code> file, or some other sort of compressed file. You'll need to right click it and uncompress it (may require downloading an application like winrar or similar if you can't see an option to decompress in the right-click menu), and then open the resulting folder.</li>
    <li>Once you've done that, you'll need to install the program that runs/executes the code. It's called <code>Node.js</code>. You should <a href="https://nodejs.org/en/download/" target="_blank">download it from here</a>. Always be careful when installing stuff on your computer. Make sure you only use the official <code>nodejs.org</code> website, otherwise you might get a virus.</li>
    <li>Once you've installed Node.js, you just need to use it to run the bot's code that you downloaded and extracted. To do that, you need to open a "command prompt" (also known as the "terminal" or "command line") inside the "app" folder that you extracted (the one with a file called <code>server.js</code> inside it). <a href="https://superuser.com/questions/339997/how-to-open-a-terminal-quickly-from-a-file-explorer-at-a-folder-in-windows-7" target="_blank">Here's how to do it in Windows</a>, and <a href="https://apple.stackexchange.com/questions/11323/how-can-i-open-a-terminal-window-directly-from-my-current-finder-location" target="_blank">here's how to do it on a Mac</a>. On Ubuntu/Linux you just right-click the folder and click "Open in terminal".</li>
    <ul>
      <li>If you're unsure whether you've opened the command prompt in the correct folder, then: if you're on <b>Windows</b>, type <code>dir</code> into the command prompt and press enter - you should see a list of files, and one of them should be called <code>server.js</code>. If you're on <b>Mac or Linux</b>, type <code>ls</code> in the command prompt and press enter. Again, one of the listed files will be called <code>server.js</code> if you're in the correct folder.</li>
      <li>If you're not sure whether you've installed node.js properly, type <code>node --version</code> in the command prompt and press enter. It should display a number that's bigger than 14. For example if it displays "v14.1.0" or a bigger number, then you've correctly installed Node.js</li>
    </ul>
    <li>Now that you've got your command prompt open in the folder, you need to type the following command into the command prompt (<i>exactly</i>) and press enter:<br><code>npm install -g pm2</code></li>
    <li>Next, install the required packages by typing <code>npm install</code> and then press enter, if there are errors/warnings/vulnerabilities/etc, run <code>npm audit fix</code> to automatically fix them.</li>
    <li>Once that has finished, type this and press enter:<br><code>pm2 start server.js</code></li>
    <li>That's it! Your bot should be working.</li>
    <li>If you close the command prompt or shut down your computer, the bot will stop. To start it up again, just go to the folder with your bot's code, open a command prompt and run this:<br><code>pm2 start server.js</code></li>
  </ol>
