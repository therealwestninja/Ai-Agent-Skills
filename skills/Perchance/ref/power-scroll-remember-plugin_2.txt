  <div class="main">
    <p>This is a very simple plugin that remembers where you scroll past a generator page, so that when you re-open the generator, the page will immediately scroll to wherever you've last scrolled to.</p>
    <p>To use this plugin, you simply put this line into the list code panel:</p>
    <pre>rememberScroll = \{import:[generatorName]\}</pre>
    <p>Then put this anywhere on the HTML panel:</p>
    <pre>\[rememberScroll()\]</pre>
    <p>To see the "remembering" effect, try scrolling to a specific part of this page (e.g., the notes), and then reloading the page, and you'll notice that the page immediately switches to that part of the page you're on before you reloaded the page (in this case, the notes).</p>
    <p>If you want the generator to save the scrolling position only for the tab session and not for the whole browser session, use this code:</p>
    <pre>\[rememberScroll("tabSessionOnly")\]</pre>
    <p>You can create a button to clear the data and make the page scroll back to the top of the page:</p>
    <pre>&lt;button onclick="rememberScroll('clearData')">Clear Data&lt;/button></pre>
    <p><button onclick="$output('clearData')">Clear Data</button></p>
    <p>Note that this will <b>reload</b> the entire page, just like the official remember plugin.</p>
    <h2>Notes</h2>
    <ul>
      <li>This plugin works in the same way as the <a href="remember-plugin">remember plugin</a>, however this is programmed to remember where you've scrolled the page instead of a specific variable/input value.</li>
