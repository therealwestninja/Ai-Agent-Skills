<h1>Upload Plugin Deletion Example</h1>
<p>This example demonstrates the <b>deletionUrl</b> parameter returned by the upload plugin. Note that deleted files may be cached for a while, so they may appear to not be deleted, but they should be gone within 24 hours or so. You can test if a file is actually deleted by appending <b>?test</b> (or any other random text start with a question mark) to the end of the file URL, and confirming that you get a 404 message. If deletion fails, then it could be because someone has uploaded an identical file, or because the deletion URL has expired (deletion is only possible within 3 days of upload).</p>
