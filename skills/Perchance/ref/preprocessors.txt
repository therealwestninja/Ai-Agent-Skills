<h1>Preprocessors ⚙</h1>
<main>
	<p style="margin-top:0">Preprocessors are functions that transform your code before it is "compiled" by the Perchance engine. They allow you to change Perchance's syntax. You could, for example, make a pre-processor that replaces all instances of <code>:smile:</code> in your Perchance code with 😊 by placing this code <b>at the top</b> of your generator:</p>
<pre>
$preprocess(text) =>
  text = text.replaceAll(":smile:", "😊");
  return text;
</pre>
  <p>Try adding that code to the top of one of your generators, and then adding :smile: to one of the list items in that generator.</p>
  <p>But that's a very simple/silly example - preprocessors are quite powerful in terms of what they allow.</p>
	<p>Here's the (very short) list of 'official' preprocessors that you can import into your generator:</p>
	<ul style="text-align:left;">
		<li><a href="/inline-dent-preprocessor">inline-dent-preprocessor</a>: allows you to use "inline" property syntax</li>
	</ul>
  <br>
  <p>To import one of the above preprocessors, simply include a line like this <b>at the top</b> of your Perchance code:</p>
<pre>
$preprocess = \{import:name-of-the-preprocessor\}	
</pre>
  <p>If you'd like to create a preprocessor that can be imported, then just create a normal generator with the <b><code>$output</code></b> as your preprocessor function like this:</p>
<pre>
<b style="color:#4d9cfb;">$output</b>(text) =>
  text = text.replaceAll(":smile:", "😊");
  return text;
</pre>
  <p>If that generator were called <code>smile-preprocessor</code> then you'd import it by adding this at the <b>top</b> of the generator where you want to use it:</p>
<pre>
$preprocess = \{import:smile-preprocessor\}	
</pre>
  <p>Below are a couple of <i>community-made</i> preprocessors for you to play around with. Note that these are saved under the admin account, so even though they're community-contributed, you don't have to worry about backwards-incompatible changes.</p>
	<ul style="text-align:left;">
		<li><a href="/inferno-shorthand-preprocessor-v1">inferno-shorthand-preprocessor-v1</a>: a type of short-hand syntax (made by <b>Infernity#3671</b> on Discord)</li>
		<li><a href="/eatham-emoji-preprocessor-v1">eatham-emoji-preprocessor-v1</a>: swaps text for emoji equivalent (made by <b>Geebs#0098</b> on Discord)</li>
	</ul>
	<br>
	<p><b>Notes/Links:</b></p>

	</ul>
