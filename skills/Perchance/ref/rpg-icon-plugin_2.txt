<h1>[$output("round-shield")] RPG Icon Plugin [$output("spiked-mace")]</h1>

<div style="margin:0 auto;width:100%;max-width:880px;background:white;border-radius:2px;padding:1em; box-sizing:border-box;">
	<p>Put this in your Perchance code panel:</p>
	<p><code>icon = \{import:rpg-icon-plugin\}</code></p>
	<br>
	<p>Now you can create any of ~500 RPG-themed icons by referencing its name:</p>
	<p><code>\[icon("sword")\]</code> → [$output("sword", "")]</p>
	<p><code>\[icon("burning-book")\]</code> → [$output("burning-book", "")]</p>
	<p><code>\[icon("explosive-materials")\]</code> → [$output("explosive-materials", "")]</p>
	<p><code>\[icon("knight-helmet")\]</code> → [$output("knight-helmet", "")]</p>
	<p><code>\[icon("axe-swing")\]</code> → [$output("axe-swing", "")]</p>
	<p><code>\[icon("dragon-breath")\]</code> → [$output("dragon-breath", "")]</p>
	<p><code>\[icon("hydra-shot")\]</code> → [$output("hydra-shot", "")]</p>
	<p>Below this plugin's explanation is a massive list of icons that you can use.</p>
	<p><b>Hover over</b> an icon in that list to get its name.<p>
	<br>
	<p>You can change the size of the icons like this:</p>
	<p><code>\[icon("sword lg")\]</code> → [$output("sword lg")]</p>
	<p><code>\[icon("sword 2x")\]</code> → [$output("sword 2x")]</p>
	<p><code>\[icon("sword 3x")\]</code> → [$output("sword 3x")]</p>
	<p>(etc. up to 5x)</p>
	<br>
	<p>You can also add custom <a href="https://en.wikipedia.org/wiki/Cascading_Style_Sheets">styling</a> to the icon:</p>
	<p><code>\[icon("dragon-breath lg", "color:red")\]</code> → [$output("dragon-breath lg", "color:red")]</p>
	<p><code>\[icon("water-drop", "color:#1b8ce8")\]</code> → [$output("water-drop", "color:#1b8ce8")] (use a <a href="https://www.google.com/search?q=hex+color+picker">hex color picker</a>)</p>
	<p><code>\[icon("sword lg", "color:white; background:black; padding:2px")\]</code> → [$output("sword lg", "color:white; background:black; padding:2px")]</p>
	<br>
	<p><b>Notes:</b></p>
	<ul>
		<li>All credit for this plugin goes to the <a href="https://github.com/nagoshiashumari/Rpg-Awesome">RPG Awesome</a> project.</li>
		<li>Check out <a href="/rpg-icon">this generator</a> which simply generates a random rpg icon (using the full list of icon names).</li>
		<li>Check out more plugins at <a href="/plugins">perchance.org/plugins</a></li>
	</ul>
</div>
