
<h1>[$output(1, true)] Perchance Logo Plugin [$output(1, true)]</h1>

<div style="margin:0 auto;width:100%;max-width:800px;background:white;border-radius:2px;padding:1em; box-sizing:border-box;">
	<p style="text-align:justify; margin-top:0;">This is a very simple little plugin that shows a perchance logo image. Just write this in your Perchance code panel:</p>
	 
<pre class="custom" style="margin:0; height:100%; box-sizing:border-box;">
plogo = \{import:perchance-logo-plugin\}
</pre>
	
  <p>and then write this in your HTML (bottom-right) panel:</p>
	
<pre class="custom" style="margin:0; height:100%; box-sizing:border-box;">
\[plogo()\]
</pre>

	<p>and it'll output a logo:</p>
	
	<p>[$output(1, true)]</p>
	
	<p>You can change the size of the logo like this:</p>
	
<pre class="custom" style="margin:0; height:100%; box-sizing:border-box;">
\[plogo(2)\]
</pre>
	
	<p>That outputs a logo that is 2x the default size:</p>
	
	<p>[$output(2, true)]</p>
	
	<p>Just change the <code>2</code> to any desired size (decimal numbers work too!).</p>

	
	<br><br>
