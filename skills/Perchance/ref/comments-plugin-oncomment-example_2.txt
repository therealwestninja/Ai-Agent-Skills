<h1>Comments Plugin onComment/onLoad Example</h1>

<p style="font-size:80%; margin:1rem auto; max-width:700px;">There's a lot you can do with this feature - you can even create multiplayer games with it! But you should <a href="https://perchance.org/simple-comments-plugin-oncomment-example#edit" target="_blank" style="font-weight:bold;">check out this more simple example</a> before looking at the code here.</p>
<p style="font-size:80%; margin:1rem auto; max-width:700px;">The onComment/onLoad functionality of the <a href="https://perchance.org/comments-plugin" target="_blank">comments-plugin</a> allows you to 'listen' for new comments and add them to your own <b>completely</b> custom HTML chat log/feed. The 'onLoad' function gives us access to the existing comments during the initial page load, and 'onComment' gives us access to the live stream of comments as they come in. What you see below is a "fake" comments list - completely custom HTML/CSS. The input area is real though (the hideComments=true option allows us to only show the input area).</p>
<p style="font-size:80%; max-width:700px; margin:1rem auto;">You can also <a href="https://perchance.org/programmatic-get-set-submit-comments-plugin-example#edit" target="_blank">programmatically submit comments on behalf of the user</a> (that's a link to a separate example).</p>
<p style="font-size:80%; margin:1rem auto; max-width:700px;">As a demonstration, I've made it so the chat bubble text color is determined by the first word in the comment. And if a comment is over a certain length, I decrease its font size. And if a comment ends with a png/jpeg/gif/webp URL, then it's turned into an image. And there are a few other features that you can see if you open the editor.</p>

<p style="font-size:90%; margin:1rem auto; max-width:700px; font-weight:bold;">Examples of stuff you can do with this:</p>
<ul style="font-size:80%; margin:1rem auto; max-width:500px;">
  <li><b>Simple:</b> Set the background image of the <i>whole page</i> to the most recent URL in the comments.</li>
  <li style="margin-top:0.5rem;"><b>Simple:</b> Make it rain confetti across the whole page when anyone says "woo!!!" in their comment.</li>
  <li style="margin-top:0.5rem;"><b>Simple:</b> Allow users to add a URL at the end of their nickname, which is hidden from their displayed nickname, but used as an avatar pic. (<a href="https://perchance.org/gqp3r7gugk" target="_blank">Example by VioneT</a> - the explanation for it is <a href="https://lemmy.world/comment/14093832" target="_blank">here</a>)</li>
  <li style="margin-top:0.5rem;"><b>Simple:</b> Implement a simple/rough "emoji reactions" feature - e.g. e.g. replies that quote another recent comment exactly (or just first few words) and have a single emoji after the quote are displayed as reaction emojis under the relevant comment instead of being displayed as an actual comment.</li>
  <li style="margin-top:0.5rem;"><b>Mad Science:</b> Allow users to play chess or tic-tac-toe with one another via the comments. User challenges another via some specific syntax (based on their user id - e.g. <b>challenge:&lt;userId&gt;</b>), then if the challenged user responds in affirmative (e.g. <b>accept:&lt;challengerUserId&gt;</b>), a game board is added to the page. Then look at their following comments for their moves (ignoring invalid moves) until the game reaches an end state. You'd likely want to use the <a href="https://perchance.org/programmatic-get-set-submit-comments-plugin-example#edit" target="_blank">programmatic comment submission</a> feature so the user doesn't have to manually copy and paste the move/turn text and click submit.</li>
  <li style="margin-top:0.5rem;"><b>Ascendent Science:</b> Make a point-and-click multiplayer adventure game. It would be <i>technically</i> possible - like tic-tac-toe and chess, but with more advanced "move"/"turn" validation rules.</li>
</ul>
